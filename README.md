# 🎮 Flux Jogos
Bem-vindo ao repositorio mais importante da nossa equipe.

## 🕹️ Como adicionar o seu jogo no catálogo

### 1. Coloque seus arquivos na pasta certa
Jogue o arquivo `.html` do seu jogo (e qualquer CSS/JS/Assets que ele precisar) dentro da pasta `jogos/`. 

### 2. Edite o arquivo `jogos.json`
Abra o `jogos.json` na raiz do projeto. Ele é a alma do nosso catálogo. Vá até o final da lista e adicione o seu jogo seguindo **exatamente** este formato:

```json
{
    "nome": "Super Mário do RH",
    "autor": "Seu Nome Aqui",
    "arquivo": "mario-rh.html",
    "imagem": "[https://link-da-sua-imagem-aqui.jpg](https://link-da-sua-imagem-aqui.jpg)"
}
```
### Em caso de investigacao: Isso aqui é um laboratório experimental de testes de interface gamificada de alta performance.
